# ソートアルゴリズム シミュレーター

React 18とTypeScript、Atomic Designパターンで構築されたソートアルゴリズムの可視化アプリケーションです。

## デモプレイ
https://yunbow.github.io/react-sim-array-sort/demo/

## 主要機能

### 1. ソート機能
- **対応アルゴリズム**
  - バブルソート
  - 選択ソート
  - 挿入ソート
  - クイックソート
  - マージソート

- **可視化要件**
  - ソート処理のステップを逐次可視化
  - 比較・交換などの状態を毎ステップ表示
  - アルゴリズム特有の概念を色分け・ハイライトで可視化
    - クイックソート：ピボット（黄色）
    - マージソート：マージ処理中の左右配列（橙色・赤色）
    - 挿入ソート：ソート済み領域（青色）
    - 選択ソート：最小値候補インデックス（橙色）

### 2. データ生成機能
- **初期データ生成パターン**
  - ランダム：ランダムに値を列挙
  - ソート済み：昇順に整列済み
  - 逆順：降順に整列済み
  - 重複が多い：同一値を多く含む
  - 山型：山の形に分布する
  - ほぼソート済み：軽微な乱れを持つデータ

- **データサイズ**
  - 10〜1000の任意のサイズを設定可能
  - デフォルトサイズ：100

### 3. ソート実行機能
- **実行制御**
  - ボタンラベル遷移：「実行」→「一時停止」→「再開」
  - リセットボタン：データ生成状態を初期化

- **ソート速度調整**
  - アニメーション速度を調整できるスライダー

### 4. アニメーション表示機能
- **バー表示**
  - データを縦棒グラフ状に表示

- **色分けルール**
  | 状態 | 色 | 説明 |
  |------|----|------|
  | 比較中 | 赤 | 比較対象の2要素 |
  | 交換発生 | 緑 | 交換対象となる2要素 |
  | 作業対象 | 橙 | 現在のループインデックス・注目している値 |
  | ピボット | 黄 | クイックソートのピボット要素 |
  | ソート完了 | 青 | 最終状態 |

### 5. UI操作機能
- ソートアルゴリズム選択プルダウン
- データパターン選択プルダウン
- データサイズ入力欄
- 実行／一時停止／再開／リセットボタン
- アニメーション速度スライダー
- **アルゴリズム比較モード**
  - 左右にアルゴリズム選択プルダウンを2つ配置
  - 初期データは完全に同一データセットを使用
  - 2つのソートを同時に実行可能

### 6. ログ・状態表示機能
- **ソート状態表示**
  - ステップ数
  - 比較回数
  - 交換回数
  - 実行時間（ミリ秒）

- **実行完了後**
  - 完了判定表示（メッセージ表示）

### 7. 計算量表示機能
- 各アルゴリズムの理論的計算量（最悪・平均・最良）をUIに表示
  - 例：バブルソート → Worst: O(n²), Average: O(n²), Best: O(n)

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - プログラミング言語
- **Storybook 7** - コンポーネント開発・ドキュメント
- **CSS Modules** - スタイリング
- **Vite** - ビルドツール

## プロジェクト構造

```
src/
├── features/                        # 機能別モジュール
│   └── sort/                        # ソート機能
│       ├── algorithms/              # ソートアルゴリズム実装
│       │   ├── bubbleSort.ts
│       │   ├── selectionSort.ts
│       │   ├── insertionSort.ts
│       │   ├── quickSort.ts
│       │   ├── mergeSort.ts
│       │   └── index.ts
│       ├── components/              # 機能専用コンポーネント
│       │   ├── BarChart/            # バーチャート可視化
│       │   ├── SortControls/        # ソートコントロール
│       │   └── SortStats/           # 統計表示
│       ├── hooks/                   # カスタムフック
│       │   └── useSortSimulator.ts
│       ├── utils/                   # ユーティリティ
│       │   └── dataGenerator.ts
│       ├── SortSimulator/           # シングルモードコンポーネント
│       └── ComparisonMode/          # 比較モードコンポーネント
├── components/                      # 共通UIコンポーネント
│   ├── Button/                      # ボタン
│   ├── Select/                      # セレクトボックス
│   ├── Slider/                      # スライダー
│   └── Input/                       # テキスト入力
├── types/                           # 汎用的な型定義
│   └── index.ts
├── App.tsx                          # メインアプリ
├── main.tsx                         # エントリーポイント
└── theme.css                        # テーマカラー定義
```
